SCP-004-LS - SCP Listenbourg

FONDATION SCP

LISTENBOURG

 

/!\ EN COURS

Objet : SCP-004-LS

Classe : Safe

Niveau de Menace : Vert

Procédures de Confinement Spéciales : SCP 4470-FR doit être confiné dans l'aile de stockage est du site phénix. Uniquement le personnel de niveau 4 est autorisé à entrer dans la cellule. 2 Caméras devrons être allumées en permanence dans la salle, si quelqu'un est détecté, la porte de la cellule doit être verrouillée, et une équipe spécialisée jugeras si la suppression de cette personne doit être exécutée.

Description : SCP 4470-FR est une machine à écrire en bois avec un clavier AZERTY en fer. La machine n'a pas de nom de marque, de numéro de série, ou même d'année de création. Cependant la machine a le style des machines à écrire des années 1900-1910. Ses capacités anormales se déclenchent quand quelqu'un met un papier dedans et marque quelque chose sur le papier. (Une feuille A4 suffit et étrangement la machine semble fonctionner avec n'importe quel type d'encre). Si quelqu'un marque une phrase sur cette machine, et qu'il retire le papier : Ce qu'il y a marqué sur le papier va se résoudre dans 100% des cas, et avec une précision de 84%. Quand la feuille est retirée la machine va faire un petit "ting" qui signifie que la "commande" est valide. Cependant le délai entre le moment où on retire le papier, et le moment où semble se réaliser la phrase est aléatoire. Si la feuille est brulée, broyée ou froissée... La "prophétie" ne se réaliseras pas. Et enfin, la machine exauceras la prophétie toujours avant la mort du sujet. (Voir test 07)

Découverte : La machine à écrire à été découverte en 1985 en Alsace. La maison dans laquelle était la machine allait se faire démolir, mais un ouvrier à trouvé la machine. Il a décidé de la garder dans le but de la revendre. Quand il s'est rendu compte de ses capacité anormales, il a voulu la commercialiser et la Fondation lui est tombée dessus. SCP 4470 FR est donc détenu par la fondation depuis une quarantaine d'années.

 

JOURNAUX DE TESTS :

- Test 1 :

 Message : "J'aimerais un café" Personne écrivant le message : Dr L████ Situation de la machine : Normale, le papier fut retiré et le tintement fût réalisé Durée du délai : 5 heures Résultat : Un café noir dans une tasse de 22cl apparut sur le bureau du chercheur`

- Test 2 :

 Message : "Un uniforme qui gratte moins" Personne écrivant le message : D-0936 Situation de la machine : Normale, le papier fut retiré et le tintement fût réalisé Durée du délai : 6 jours et 17 heures Résultat : Un nouvel uniforme apparu dans la cellule de D-0936. Il a été confisqué en étant qualifié de : Objet provenant d'une anomalie.

- Test 3 :

 Message : Le cœur a ses raisons que la raison ne connaît pas. Personne écrivant le message : D-0936 Situation de la machine : Normale, mais aucun tintement fut réalisé Durée du délai : Inconnu Résultat : Aucun Note : SCP 4470 FR ne fonctionne qu'avec des "commandes" les citations ne doivent pas fonctionner

- Test 4 :

 Message : "Un peu d'air, qui viens de ma campagne" Personne écrivant le message : D-4526 Situation de la machine : Normale, le papier fut retiré et le tintement fût réalisé Durée du délai : 30 secondes Résultat : Le sujet à rapidement déclaré que "WOW, cette machine est trop bien, j'sens l'air de ma campagne !"

- Test 5 :

 Message : "Des amis" Personne écrivant le message : Dr ██████ Situation de la machine : Normale, le papier fut retiré et le tintement fût réalisé Durée du délai : 1 an et 5 mois Résultat : Le Dr ██████ à déclaré qu'il s'était fait un vrai ami Note : SCP 4470 FR peut créer des liens sociaux, cela m'as donné une idée d’expérience...

- Test 6 :

 Message : "Rendre D-9264 violent" Personne écrivant le message : D-9264 Situation de la machine : Normale, le papier fut retiré et le tintement fût réalisé Durée du délai : 4 minutes Résultat : Pendant se reconduite a sa cellule, D-9264 a subitement attaqué les gardes qui l'escortait, ceci a résulté à la suppression de D-9264

- Test 7 :

 Message : "Tuer SCP 682" Personne écrivant le message : D-0936 Situation de la machine : D-0936 a commencé a taper le message mais arrivant au "2" la machine émit un craquement, et la touche "2" ne répondit plus. Durée du délais : Inconnu Résultat : Aucun Note : Après analyse, le mécanisme s'occupant de la touche "2" a complètement arrêté de fonctionné, selon un spécialiste, une réparation de cette touche risque de détruire le mécanisme du clavier. Nous allons donc laisser SCP 4470 FR comme ceci. Note du chercheur en charge de SCP 4470 FR : "Après de différent tests, nous avons constatés, que SCP 4470 FR exauce toujours la phrase **avant** la mort du sujet, le test 08 a donc été mis au point".

- Test 8 :

 Consigne post-test : Supprimer D-4229 juste après le tintement de SCP 4470 FR. Des caméras analysant a la micro seconde ont été placés devant le sujet, afin de vérifier le moment exact de la réalisation du message. Message : "Raser mon sourcil droit" Personne écrivant le message : D-4229 Situation de la machine : D-4229 à marqué la message, puis a retiré le papier, le tintement fut réalisé. L'agent P████ supprima D-4229 afin de réaliser le test. Durée du délais : 0,316415 secondes Résultat : La moitié du sourcil droit de D-4229 se supprima avant le coup de feu de L'agent P████
